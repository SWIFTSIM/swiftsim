Minimal (Density-Entropy) SPH
=============================

.. toctree::
   :maxdepth: 2
   :hidden:
   :caption: Contents:

This scheme is a textbook implementation of Density-Entropy SPH, and can
be used as a pedagogical example. It also implements a Monaghan AV scheme
and Balsara switch, like the GADGET-2 scheme. It is effectively the same
equations, but implemented differently.
