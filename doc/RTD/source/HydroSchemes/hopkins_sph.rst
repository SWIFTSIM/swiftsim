Pressure-Entropy SPH
====================

.. toctree::
   :maxdepth: 2
   :hidden:
   :caption: Contents:

A pressure-entropy SPH scheme is available in SWIFT, inspired by Hopkins 2013.
This includes a Monaghan AV scheme and a Balsara switch.


.. code-block:: bash
   
   ./configure --with-hydro="hopkins"
